# k6 Benchmark Runbook

## Command

```bash
k6 run k6/track_benchmark.js \
  -e TRACK_URL=http://localhost:8018/track \
  -e K6_VUS=200 \
  -e K6_DURATION=25s \
  --summary-export docs/benchmarks/k6_summary.json
```

## Notes
- Target request volume: approximately 5,000 under 200 VUs (depends on host performance).
- For webhook failure injection validation, set `-e FAIL_WEBHOOK=true` and subscribe consumers to failing target URLs.
- Raw machine-readable output is written to `docs/benchmarks/k6_summary.json`.
- Do not claim benchmark metrics unless this file is generated by an actual run.
